ZyberAuto
=========

Android app project for ZyberAuto.

Overall Folder Overview
-----------------------
The repository root contains the Android Studio project, with the main app module under app/ and build tooling/configuration files at the root. Generated build outputs may appear under app/build/ and build/.

Project Structure
-----------------
- app/: Android application module
- build.gradle.kts / settings.gradle.kts: Gradle build configuration

Requirements
------------
- Android Studio (latest stable)
- Android SDK + Platform Tools
- JDK 17 (recommended for modern Android builds)

Build & Run
-----------
Open the project in Android Studio and run the app from the IDE.

Notes
-----
If you encounter Gradle sync issues, verify that your SDK and JDK paths are configured in Android Studio settings.
