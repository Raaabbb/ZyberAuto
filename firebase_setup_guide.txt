FIREBASE SETUP GUIDE FOR ZYBERAUTO
====================================

STEP 1: CREATE FIREBASE PROJECT
-------------------------------
1. Go to https://console.firebase.google.com/
2. Click "Create a project" (or "Add project").
3. Name it "ZyberAuto".
4. Toggle Google Analytics OFF (optional, simplifies setup) and click "Create Project".
5. Wait for it to finish and click "Continue".

STEP 2: REGISTER ANDROID APP
----------------------------
1. On the Project Overview page, click the Android icon (the little robot).
2. Enter these details:
   - Android package name: com.example.zyberauto
   - App nickname: ZyberAuto (Optional)
   - Debug signing certificate SHA-1: (Leave blank for now)
3. Click "Register app".

STEP 3: ADD CONFIG FILE
-----------------------
1. Download the "google-services.json" file provided by Firebase.
2. Move this file into your project's "app" folder.
   Path: ZyberAuto\app\google-services.json

STEP 4: UPDATE GRADLE SCRIPTS
-----------------------------
You need to add the Google Services plugin dependencies.

A. Open `build.gradle.kts` (Project Level - Root folder)
   Add this to the `plugins` block if it's not there (check the version on Firebase docs, usually 4.4.x):
   
   id("com.google.gms.google-services") version "4.4.0" apply false

B. Open `app/build.gradle.kts` (Module Level - App folder)
   1. In the `plugins` block at the top, add:
      id("com.google.gms.google-services")

   2. In the `dependencies` block, add the Firebase BoM (Bill of Materials) and dependencies:
      // Import the Firebase BoM
      implementation(platform("com.google.firebase:firebase-bom:32.7.0"))
      
      // Add the dependencies for the Firebase products you want to use
      // When using the BoM, don't specify versions in Firebase library dependencies
      implementation("com.google.firebase:firebase-analytics")
      implementation("com.google.firebase:firebase-auth")
      implementation("com.google.firebase:firebase-firestore")

STEP 5: SYNC AND VERIFY
-----------------------
1. Click "Sync Now" in the notification bar in Android Studio (top right).
2. Once synced, build the app to make sure there are no errors.
3. Back in the Firebase Console, click "Next" until you reach the end of the setup wizard.

TROUBLESHOOTING
---------------
- If you get a "Plugin not found" error, make sure the version in the Project Level gradle file matches what is available in Google Maven repository.
- Ensure the package name in `google-services.json` matches `package="com.example.zyberauto"` in your AndroidManifest.xml or `namespace` in app/build.gradle.kts.
